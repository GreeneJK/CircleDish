# Save Circle Dish Logo

## Quick Steps:

1. **Save the logo image** you uploaded to:
   ```
   /Users/jeremygreene/CascadeProjects/CircleDish/frontend/public/images/circle-dish-logo.png
   ```

2. **The logo should be**:
   - The image with "CircleDish" text and whisk icon
   - "COOK. SHARE. DISCOVER." tagline
   - PNG format (transparent background preferred)
   - Approximately 800x240px or similar ratio

3. **After saving**, the logo will appear:
   - In the header navigation (smaller version)
   - In the hero section of the landing page (larger version)

## Alternative: Use SVG (Recommended)

If you have an SVG version of the logo, save it as:
```
frontend/public/images/circle-dish-logo.svg
```

Then update the Image components to use `.svg` instead of `.png`

## Verify

After saving, run:
```bash
ls -lh frontend/public/images/circle-dish-logo.*
```

You should see the logo file listed.
